# Codechef Albums

Create albums and share it with users or make it private.

### Prerequisites


```
php
mysql
apache/xamppp
```

## Built With

* [PHP](http://php.net) - PHP
* [MySQL](https://www.mysql.com) - MySQL
* [Semantic-UI](https://semantic-ui.com) - Semantic UI
